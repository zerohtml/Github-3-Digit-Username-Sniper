# 🚀 GitHub 3-Letter Username Sniper

This tool automatically scans and claims available 3-letter usernames on GitHub. / Bu araç, boşta olan 3 harfli GitHub kullanıcı adlarını otomatik olarak tarar ve yakalar.

---

## 🇺🇸 English Guide

### 🔑 How to Get Your GitHub Token?
1. Go to **Settings** > **Developer Settings**.
2. Select **Personal access tokens** > **Tokens (classic)**.
3. Click **Generate new token (classic)**.
4. Name it "Sniper" and set expiration as you wish.
5. **IMPORTANT:** Check the **`user`** (Update public profile data) box.
6. Click **Generate token** and copy the code.

### 🛠️ Installation & Usage
1. Install requirements: `pip install requests`
2. Open `sniper.py` and paste your token into `GITHUB_TOKEN`.
3. Run: `python sniper.py`

## 🇹🇷 Türkçe Rehber

### 🔑 GitHub Token Nasıl Alınır?
1. **Settings** (Ayarlar) > **Developer Settings** kısmına gidin.
2. **Personal access tokens** > **Tokens (classic)** seçeneğine tıklayın.
3. **Generate new token (classic)** butonuna basın.
4. Bir isim verin (Örn: "Sniper") ve süresini belirleyin.
5. **ÖNEMLİ:** Yetkiler arasından **`user`** (Update public profile data) kutucuğunu işaretleyin.
6. **Generate token** butonuna basın ve çıkan kodu kopyalayın.

### 🛠️ Kurulum ve Kullanım
1. Kütüphaneyi yükleyin: `pip install requests`
2. `sniper.py` dosyasını açın ve token'ınızı `GITHUB_TOKEN` kısmına yapıştırın.
3. Çalıştırın: `python sniper.py`
